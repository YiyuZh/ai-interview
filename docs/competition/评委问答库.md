# 评委问答库

## Q1：你是不是只是套大模型 API？
不是。底层语言能力来自大模型，但系统核心是岗位画像、简历证据、能力差距、简历润色、追问、报告、学习任务、人工评分、Eval 和 SFT Preview 的工程闭环。

## Q2：简历润色会不会帮学生造假？
不会。简历润色 Agent 只强化已有证据。缺失能力只能提示补证据，不会编造项目、公司、时间、指标或技术栈。

## Q3：真实微调完成了吗？
当前完成的是 SFT-ready 数据结构、Preview 生成链路和 OpenAI SFT 启动脚本。只有真实 job 成功并拿到 fine_tuned_model 后，才能说完成真实微调。

## Q4：演示样本是真实用户吗？
不是。演示样本是 `demo_constructed`，`for_training=false`，只用于比赛展示和链路验证。

## Q5：为什么做 Eval？
Eval 把“效果更好”变成可评分维度。赛前是 Preview，后续会用真实授权 holdout 样本做正式对比。
