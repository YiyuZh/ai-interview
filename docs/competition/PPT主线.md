# 职启智评 Career-AgentOS 答辩 PPT 主线

## 一句话主张
职启智评不是随机 AI 面试官，而是一个基于 Career-AgentOS 的多 Agent 就业能力诊断与训练平台。

## 叙事顺序
1. 大学生求职痛点：岗位认知弱、简历证据弱、面试训练弱。
2. 项目定位：从简历解析到能力诊断、简历润色、模拟面试、报告和学习任务。
3. 技术升级：Career-AgentOS 多 Agent 编排。
4. 核心链路：简历证据 -> 岗位画像 -> 能力差距 -> 简历润色 -> 面试追问 -> 报告 -> 学习任务。
5. 演示资产：三岗位沙盘、Agent Trace、Eval Preview、SFT Preview。
6. 数据闭环：授权、去标识化、人工评分、样本准入。
7. 后训练路线：OpenAI SFT、DPO、QLoRA 三阶段补实。

## 答辩边界
- 可以说：已完成多 Agent 展示闭环、SFT Preview 和 Eval Preview。
- 不能说：已完成真实 OpenAI 微调、已有 fine_tuned_model、构造样本是真实用户。
