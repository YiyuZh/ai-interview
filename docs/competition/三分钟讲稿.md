# 三分钟讲稿

各位评委老师好，我们的项目是职启智评，一个面向大学生求职准备的 Career-AgentOS 多智能体就业能力诊断与训练平台。

传统 AI 面试工具通常只是随机生成问题，学生不知道简历证据是否足够，也不知道岗位到底看什么。我们把流程拆成多个 Agent：简历证据 Agent 先判断能力是否有真实证据，岗位画像 Agent 给出目标岗位要求，能力差距 Agent 找出缺口，简历润色 Agent 只基于真实证据给岗位化表达建议，面试追问 Agent 再围绕缺口追问，最后生成报告、学习任务，并通过用户授权和后台人工评分沉淀样本。

这次比赛版我们重点做了可追溯展示：三岗位沙盘案例、Agent Trace、Eval Preview 和 SFT Preview。评委可以看到每一步为什么这样判断、为什么这样润色、为什么这样追问。需要强调的是，当前 demo 样本是比赛演示数据，`for_training=false`，不冒充真实用户样本；真实 OpenAI SFT 会在授权样本和人工复核达标后补实。

我们的创新点是把就业诊断从一次性大模型回答，升级为证据约束、多 Agent 协作、可评估、可沉淀、可后训练的 AI 工程系统。
