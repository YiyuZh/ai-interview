## 9. 命名规范

```text
演示数据：demo_*
预览文件：*.preview.*
真实训练文件：不带 preview，并必须有 summary 中 ready_for_openai_job=true
Trace 文件：*.trace.json / *.trace.md
Eval 文件：*.eval.csv / eval_summary.md
```

## 10. 状态字段规范

```text
sample_origin:
  demo_constructed       比赛演示构造样本
  constructed            普通构造样本
  real_authorized        真实授权样本
  imported_public        公开样例，需注明来源

for_training:
  true / false

for_competition_demo:
  true / false

review_status:
  pending
  human_reviewed
  rejected
  approved_for_eval
  approved_for_training
```
