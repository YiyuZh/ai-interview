# Career-AgentOS 演示脚本

## 演示入口
- 用户端：`/competition/agent-trace`
- 兼容入口：`/competition-demo`
- 后台：`/evaluation-datasets` 的“比赛 Preview 展示层”

## 演示顺序
1. 打开 `/competition/agent-trace`。
2. 选择 Python 后端案例。
3. 展示样本来源：`demo_constructed`、`for_training=false`。
4. 展示简历证据链。
5. 展示能力差距。
6. 展示简历润色 Agent：可改但不造假。
7. 展示面试追问。
8. 展示 Eval 七维评分。
9. 展示 SFT Preview 摘要。
10. 切换产品助理和人力资源案例，说明跨岗位能力。

## 关键话术
“这是沙盘演示数据，不是伪装成真实用户数据。我们用它验证链路和展示评审可见的技术闭环，真实训练会使用授权和人工复核样本。”
