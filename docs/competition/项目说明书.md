# 职启智评项目说明书

## 1. 项目名称

职启智评：基于大语言模型与岗位画像匹配的大学生求职能力评估及模拟面试系统

## 2. 项目背景

大学生在求职准备中常见问题包括简历表达空泛、岗位目标不清、面试回答缺少证据、改进建议不够具体。传统就业指导依赖人工咨询，难以为每位学生提供高频、个性化、可复盘的训练。

职启智评以高校就业指导为应用场景，在原有多岗位 AI 模拟面试主流程上增加岗位画像、轻量匹配评分、证据追问和人工评分对比，将通用大语言模型能力转化为结构化、可解释、可复核的求职训练流程。

## 3. 核心功能

- 简历解析：上传 PDF 简历，提取教育经历、实践经历、技能证书、项目经历等结构化信息。
- 岗位画像：为 Python 后端、Java 后端、前端、测试、算法、数据分析、产品、HR、运营、行政等岗位建立核心能力、典型任务和面试知识点。
- 匹配分析：结合目标岗位和简历证据，输出关键词覆盖率、TF-IDF 语义分、规则评分、优势、短板和改进建议。
- 模拟面试：根据简历和岗位画像生成问题，并围绕证据不足处追问。
- 能力报告：输出综合评分、分题表现、训练建议和证据闭环摘要。
- 人工评分对比：后台标注高质量样本，导出评测数据，用于答辩验证。

## 4. 技术路线

```text
Vue 用户端 / Vue 后台端
    ↓
FastAPI 接口层
    ↓
简历解析、岗位画像、轻量匹配评分、模拟面试、报告生成
    ↓
DeepSeek / OpenAI 等大语言模型 API
    ↓
PostgreSQL、Redis、Celery、Docker 部署
```

## 5. 创新点

1. 保留多岗位模拟面试生态，技术岗与非技术岗都能使用岗位画像增强。
2. 用岗位画像、八股知识点和评分规则约束 AI 输出，避免普通开放式问答的随意性。
3. 引入关键词覆盖率、TF-IDF/余弦相似度和规则评分，形成可解释匹配指标。
4. 引入证据追问闭环，优先核实简历中缺证据或高风险表述。
5. 支持后台人工标注和评测集导出，为 AI 与人工评分对比及后续微调准备数据。

## 6. 边界声明

本项目当前阶段不训练底层大模型，不声称自研大模型，不声称真实商业落地，也不把微调作为已完成成果。项目创新集中在应用层：多岗位画像、知识切片、轻量机器学习匹配、规则评分、Prompt 约束、证据闭环、可解释报告和系统工程实现。

## 7. 后续展望

- 扩大真实样本规模，积累更多简历、岗位和人工评分数据。
- 引入轻量评分校准模型，提升 AI 评分与人工评分的一致性。
- 与高校就业指导流程结合，形成学生求职能力成长档案。
- 在数据充分且合规的前提下，探索领域微调或本地模型部署。
