# 复赛限时实测 SOP

## 1. 读题 5 分钟

- 场景是什么。
- 用户是谁。
- 输入材料是什么。
- 输出物是什么。
- 风险边界是什么。

## 2. 拆流程 10 分钟

用固定框架：

```text
输入 -> 画像 -> 证据 -> 缺口 -> 生成 -> 复核 -> 任务 -> 数据治理 -> Eval
```

## 3. 分配 Agent 10 分钟

列出人、ChatGPT、Codex、业务大模型、Eval/后台评分分别做什么。

## 4. 生成交付 20-30 分钟

输出一页方案、一个流程图、一个分工表、一个风险边界和一个三个月路线。

## 5. 讲解 5 分钟

只讲主线：问题、工作流、人机边界、可运行底座、补实路线。
