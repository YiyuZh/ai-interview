# Prompt 模板库

## 1. 赛事分析 Prompt

```text
请基于官方赛事通知，提炼 OPC 赛道评审重点，并把职启智评映射到 AI 原生结构、人机协同、垂直场景和轻量化创新单元四个维度。
```

## 2. Codex 任务 Prompt

```text
请在 D:\apps\ai-interview 中执行本轮任务。先读 PROJECT_MEMORY.md、任务记录文档和升级流程手册，只修改本轮相关文件，不使用 git add .，完成后给出测试和提交信息。
```

## 3. 业务大模型 Prompt

```text
请基于目标岗位和候选人简历证据状态生成追问。岗位知识库只能作为要求和追问方向，不能写成候选人真实经历。
```

## 4. Eval Prompt

```text
请按岗位贴合、证据约束、追问深度、润色边界、格式稳定、报告可用性、隐私合规评分，并指出是否可以进入训练样本。
```

## 5. 答辩转译 Prompt

```text
请把技术实现转成 OPC 答辩语言，突出一个人如何用 AI Agents 扩展为轻量化创新单元，同时保留不能夸大的边界。
```
