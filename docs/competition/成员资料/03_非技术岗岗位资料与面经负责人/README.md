# 03 非技术岗岗位资料与面经负责人

## 负责范围

- 产品助理、运营助理、新媒体运营、人力资源专员、招聘助理、行政管理助理等非技术岗。
- 搜集和整理岗位要求、场景任务、企业侧重点、面试问题、参考回答要点和复盘反思。

## 资料应该放什么

- 非技术岗招聘 JD 的硬性要求和加分项归纳。
- 场景题、沟通题、流程题、数据复盘题和案例题。
- 公司侧重点，例如产品岗重 PRD 和指标，运营岗重活动复盘，HR 重招聘流程和沟通判断。
- 来源审核记录。

## 合格资料标准

- 不写空泛评价，必须能落到具体场景、任务、能力和验收方式。
- 非技术岗资料不要硬套“写 Demo”的口径。
- 每条面经建议拆成真实问题、参考回答要点、复盘反思、考察能力。
