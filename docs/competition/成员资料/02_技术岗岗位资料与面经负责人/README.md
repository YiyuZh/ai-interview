# 02 技术岗岗位资料与面经负责人

## 负责范围

- Python 后端、Java 后端、前端开发、测试工程师、算法工程师、数据分析师等技术岗。
- 搜集和整理岗位要求、企业侧重点、真实面试问题、参考回答要点、复盘反思和来源审核。

## 资料应该放什么

- 技术岗招聘 JD 的硬性要求和加分项归纳。
- 技术岗公开面经方向归纳。
- 技术问题的参考回答要点，不直接复制原文。
- 企业侧重点，例如大厂更看重系统设计、项目排障、数据库和缓存一致性。
- 来源审核记录：可入库、仅参考、待核验、不采用。

## 合格资料标准

- 至少写清岗位、技术主题、来源、访问日期和适用能力项。
- 面经资料必须改写为“考察方向 + 参考回答要点 + 复盘反思”。
- 不把来源不明帖子直接当正式题库。
