## 研究内容

本项目面向大学生求职准备与高校就业指导场景，设计并实现“职启智评”多岗位求职能力评估及模拟面试系统。系统基于大语言模型 API、岗位画像库、轻量机器学习匹配、结构化评分规则和证据约束机制，完成简历解析、岗位匹配分析、模拟面试问答、回答质量评价和求职能力报告生成。

项目首批构建 Python 后端、Java 后端、前端开发、测试工程师、算法工程师、数据分析师、产品助理、人力资源专员、招聘助理、运营助理、新媒体运营、行政管理助理等岗位画像。技术岗画像补充语言基础、数据库、Redis、HTTP、并发、算法、系统设计、项目追问和线上排障等知识点；非技术岗画像补充典型任务、场景问题、沟通协作和数据复盘要求。系统围绕岗位匹配度、专业能力、实践经历、表达逻辑、完整度和改进潜力六个维度进行评价，并输出关键词覆盖率、TF-IDF 语义分、规则评分和最终匹配分。系统支持后台人工标注和评测样本导出，用于开展 AI 评分与人工评分对比，提升评价结果的可解释性和可复核性。

本项目当前不训练底层大模型，不宣称自研大模型，也不宣称已完成真实微调。项目重点探索通用大语言模型、embedding/语义匹配、规则评分和证据约束在高校就业指导场景中的应用创新，并通过人工评分样本和 JSONL 导出为后续微调准备数据。
