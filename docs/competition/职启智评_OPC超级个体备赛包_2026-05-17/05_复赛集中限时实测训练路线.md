# 复赛集中限时实测训练路线

## 1. 为什么必须提前练

OPC 专项复赛不是传统路演，它会集中考察：

```text
AI 工作流构建
人机协同逻辑设计
产业应用结构表达
阶段成果展示
```

所以你不能只背 PPT。你要练“现场拿到题目，如何快速构建 AI 工作流”。

## 2. 训练总目标

形成一套固定反应模式：

```text
看题 -> 拆场景 -> 定人机边界 -> 设计 Agent -> 画流程 -> 给交付物 -> 讲治理和风险
```

## 3. 30 分钟实测题模板

### 题目示例

```text
请针对一个新的大学生求职岗位，例如“数据分析实习生”，设计一套 AI 原生就业训练工作流，并说明人和 AI 如何分工。
```

### 你的现场作答结构

```text
1. 场景痛点：岗位认知不清、简历证据不足、面试训练少。
2. 输入：简历、目标岗位、岗位画像、历史问答、学习路线。
3. Agent：岗位画像 Agent、证据评分 Agent、Gap Agent、面试追问 Agent、报告 Agent、学习任务 Agent。
4. 人机边界：AI 做检索和生成，人做资料审核、授权、评分和最终判断。
5. 输出：诊断报告、学习任务、面试追问、复盘建议、数据沉淀。
6. 风险控制：不编造经历、去标识化、人工评分、Eval。
```

## 4. 45 分钟实测题模板

### 题目示例

```text
请说明你如何用 AI 工具把一个人扩展成一个轻量化创新单元。
```

### 你的现场作答结构

```text
1. 我本人是 OPC Commander，负责目标、边界、验收、责任。
2. Codex 是技术执行 Agent，负责代码、部署、测试、日志。
3. ChatGPT 是策略和材料 Agent，负责赛事研究、PPT、讲稿、Q&A。
4. 业务大模型是服务 Agent，负责简历、岗位、面试和报告。
5. 数据治理 Agent 负责授权、脱敏、人工评分和样本准入。
6. 最终形成一个人可运营的高校就业服务单元。
```

## 5. 60 分钟实测题模板

### 题目示例

```text
请基于你的项目，设计三个月产业应用验证计划。
```

### 你的现场作答结构

```text
第 1 月：真实样本和高校场景验证
- 完成 C1/C2/C3 真实闭环
- 补 10 条授权样本
- 后台人工评分

第 2 月：模型和工作流优化
- OpenAI SFT 或 Prompt/RAG 对照
- Eval 评估
- 修复证据约束和报告质量

第 3 月：场景对接和OPC治理
- 小范围白名单试用
- 生成高校就业服务方案
- 输出数据治理和主体责任说明
```

## 6. 练习评分表

每次练习按 100 分打分：

| 维度 | 分值 | 高分标准 |
|---|---:|---|
| 场景定位 | 15 | 明确高校就业服务问题，不泛泛讲 AI |
| 工作流构建 | 25 | 输入、处理、输出、反馈闭环清晰 |
| 人机协同 | 20 | 人和 AI 分工边界明确 |
| 轻量化创新 | 15 | 能说明一个人如何替代小团队工作量 |
| 风险治理 | 15 | 隐私、幻觉、授权、人工评分说得清 |
| 表达清晰 | 10 | 3 分钟内能讲完整 |

## 7. 每天训练安排

```text
Day 1：背熟 OPC 主叙事和人机分工。
Day 2：练 30 分钟新岗位工作流设计。
Day 3：练 45 分钟一人公司治理结构。
Day 4：练 60 分钟三个月补实计划。
Day 5：让 ChatGPT 模拟评委连续追问。
Day 6：录屏复盘，修正废话和不稳表达。
Day 7：压缩成 3 分钟、8 分钟、15 分钟三个版本。
```
