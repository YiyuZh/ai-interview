# 04 ChatGPT 提示词库：按阶段

## 1. 赛事规则解读 Prompt

```text
你是一名熟悉大学生创新创业赛事、OPC 超级个体、人机协同结构和 AI 产品答辩的竞赛顾问。
我正在报名第五届“京彩大创”北京大学生创新创业大赛的“超级个体（OPC）”方向。
请你基于赛事原文和我的项目资料，判断评委真正考察什么。

要求：
1. 区分产品能力、AI 工作流能力、人机协同能力、产业嵌入能力、轻量化创新能力。
2. 判断我的项目原本哪里不对口，应该如何重构参赛叙事。
3. 输出能直接写进 PPT 的核心表达。
4. 输出后续给 Codex 的工程改造方向。
```

## 2. 项目资料理解 Prompt

```text
你是一名 AI 产品架构师和比赛答辩技术顾问。
请阅读我上传的任务记忆文档、项目升级流程手册、AI 原生结构说明书和 PPT 状态汇总。

请输出：
1. 当前项目真实具备哪些能力。
2. 哪些能力适合包装成 OPC AI 工作流。
3. 哪些内容只是 preview 或待补实，不能说成真实结果。
4. 如何把项目从“AI 面试平台”改讲成“一人 + 多 AI Agents 的就业服务单元”。
5. 请生成 Codex 可执行任务清单。
```

## 3. AI 协同链设计 Prompt

```text
你现在不是帮我介绍产品，而是帮我设计 OPC 参赛中的“AI 协同指挥链”。

请围绕以下问题输出：
1. 我作为 OPC 总控者，先如何使用 ChatGPT？
2. ChatGPT 输出什么给 Codex？
3. Codex 如何读取任务记忆、理解项目约束、生成代码和材料？
4. 业务大模型在产品内部承担哪些任务？
5. Eval / 后台人工评分如何校验 AI 输出？
6. 每轮聊天记录如何转化成 Codex 任务包？
7. 每个 prompt 的侧重点是什么？
8. 复赛现场限时实测时，我应该如何展示这种 AI 协同能力？

要求：给出流程图、Prompt 模板、Codex 任务模板、答辩讲法、文件结构。
```

## 4. Codex 任务包生成 Prompt

```text
请把以上分析整理成一份 Codex 可直接执行的任务包。
要求：
1. 使用“你是 Codex 工程 Agent”开头。
2. 写清必须先读哪些项目记忆文件。
3. 写清新增目录和文件。
4. 写清 P0/P1/P2 任务。
5. 写清不能做什么。
6. 写清验证方式。
7. 写清任务记忆文档怎么追加。
```

## 5. 答辩转译 Prompt

```text
以下是 Codex 本轮新增/修改文件、测试结果和未完成事项。
请把它转化成 OPC 答辩表达。

请输出：
1. 这轮工作证明了什么 AI 工作流能力。
2. 如何用 30 秒讲给评委听。
3. 如何用 PPT 一页展示。
4. 评委可能追问什么，如何回答。
5. 哪些话可以强势说，哪些话不能夸大。
```
