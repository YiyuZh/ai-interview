# 08 复赛现场限时实测 SOP

## 目标

现场拿到陌生任务时，展示你能快速构建 AI 工作流，而不是只会问一句 AI。

## 30 分钟流程

### 0-3 分钟：人工读题

判断：

```text
任务场景
目标用户
输入输出
AI 可做什么
人必须判断什么
风险边界
```

### 3-8 分钟：问 ChatGPT 拆工作流

```text
你是 OPC AI 工作流设计顾问。
现场任务是：{题目}
请在 5 分钟内拆成：
1. 场景问题
2. 人机协同结构
3. AI Agent 分工
4. 工作流步骤
5. 输入输出
6. 风险边界
7. Codex 任务包
```

### 8-15 分钟：让 ChatGPT 生成 Codex 任务

```text
请把刚才的工作流设计转成 Codex 可执行任务包：
1. 要生成的 Markdown 文件
2. 要生成的 JSON 样例
3. 可选前端展示页
4. 不允许做的事
5. 验收标准
```

### 15-23 分钟：交给 Codex 或展示 SOP

能用 Codex 就执行任务包；不能用 Codex 就展示已有 `Codex执行SOP` 和样例资产，说明同类任务如何落地。

### 23-28 分钟：生成答辩话术

```text
请把这个 AI 工作流用 60 秒讲给评委听，突出我作为 OPC 如何调度 AI、AI 之间如何协作、为什么一个人能完成小团队工作。
```

### 28-30 分钟：口头总结

```text
我先判断任务边界；
再让 ChatGPT 做结构拆解；
再把结果交给 Codex 生成文档/脚本/页面；
再由业务大模型执行垂直场景内容生成；
最后通过 Eval 和我本人验收。
```
