# 11 AI 协同证据材料整理规范

## 证据类型

| 证据 | 说明 |
|---|---|
| Prompt 链 | 从 ChatGPT 分析到 Codex 任务再到验收的连续 Prompt |
| Codex 任务包 | ChatGPT 输出给 Codex 的结构化任务 |
| 任务记忆记录 | 每轮处理的背景、决策、验证、未完成事项 |
| Agent Trace | 系统内部多 Agent 处理一个案例的过程 |
| Eval Preview | 用评分维度比较 baseline 与优化版 |
| SFT Preview | 样本准备链路，不冒充真实训练 |
| Git 提交记录 | Codex 落地代码和材料的痕迹 |
| 构建/测试记录 | 证明不是纯 PPT |

## 截图清单

```text
1. ChatGPT 分析 OPC 赛道的聊天截图
2. ChatGPT 生成 Codex 任务包的截图
3. Codex 读取 PROJECT_MEMORY 并执行任务的截图
4. Codex 输出新增/修改文件清单的截图
5. Git diff 或 git log 截图
6. /competition/agent-trace 页面截图
7. Eval Preview 页面截图
8. SFT Preview summary 截图
9. 任务记录文档阶段追加截图
```
