# 通俗手册：原项目改了什么，创新点在哪里

## 原项目
原项目已经有简历解析、岗位画像、能力诊断、简历润色、模拟面试、报告、学习任务、训练复盘和后台人工评分。

## 比赛版升级
比赛版不是重写项目，而是在原有功能上增加 Career-AgentOS 展示层：

- 把每个关键步骤拆成 Agent。
- 用 Agent Trace 展示推理过程。
- 用 Eval Preview 展示输出质量。
- 用 SFT Preview 展示后续微调数据结构。
- 用简历润色 Agent 强调“岗位化表达但不造假”。

## 创新点
1. 多 Agent 就业诊断架构。
2. 简历证据链约束。
3. 岗位画像驱动追问。
4. 证据约束简历润色。
5. 人类在环评分和数据治理。
6. SFT、DPO、QLoRA 后训练补实路线。

## 边界
当前比赛演示样本不是真实用户数据，SFT Preview 不是已完成真实微调。
