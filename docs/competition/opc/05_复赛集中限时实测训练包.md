# 05 复赛集中限时实测训练包

## 1. 训练目标

复赛实测不只是演示现有系统，而是证明参赛者能在限时条件下把一个新问题拆成 AI 工作流，并说明人机边界和验收标准。

## 2. 三类训练题

| 题目 | 限时 | 目标 |
|---|---:|---|
| 构建 AI 工作流 | 30 分钟 | 给一个新岗位，拆成可执行 Agent 流程 |
| 重构人机协同结构 | 45 分钟 | 说明人做什么、AI 做什么、如何验收 |
| 垂直场景落地方案 | 60 分钟 | 把职启智评迁移到一个相近服务场景 |

详细题目见 `live_test_drills/`。

## 3. 固定作答框架

```text
1. 识别场景和用户。
2. 拆解业务流程。
3. 标注人机边界。
4. 分配 AI Agent。
5. 定义输出物。
6. 设置质量和合规门禁。
7. 给出三个月补实路线。
```

## 4. 现场话术

```text
我不会直接让 AI 自由生成最终答案，而是先定义目标和边界，再把任务拆给策略、工程、业务和评估 Agent。每个 Agent 只负责自己的环节，最后由我进行真实性、合规性和可执行性验收。
```
