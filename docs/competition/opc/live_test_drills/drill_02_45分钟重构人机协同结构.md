# Drill 02：45 分钟重构人机协同结构

## 题目

评委要求你解释“一个人 + AI”如何替代传统就业服务小团队。请在 45 分钟内重构人机协同结构。

## 限时

45 分钟。

## 输入材料

- 当前项目功能清单。
- 传统小团队角色：产品、研发、就业顾问、简历顾问、测试、运营、材料负责人。

## OPC 拆解

1. 列出传统团队角色。
2. 映射到 OPC 中的人和 AI。
3. 标注人类不可外包的判断。
4. 标注 AI 可承担的高频执行。
5. 写出失败兜底。
6. 写出答辩一句话。

## AI Agent 分工

- OPC Commander Agent：确定最终人机边界。
- ChatGPT 策略 Agent：生成协同结构表达。
- Codex Engineering Agent：指出真实工程证据。
- Data Governance Agent：检查隐私和训练数据边界。
- Defense Coach Agent：生成评委追问回答。

## 最终交付物

- “小团队角色 -> OPC 替代方式”表格。
- “人负责 / AI 负责 / 兜底机制”表格。
- 90 秒答辩说明。

## 评分点

- 是否体现 OPC，而不是普通团队项目。
- 是否说明最终责任仍在人。
- 是否结合职启智评真实功能。
- 是否避免“AI 替代一切”的危险表达。

## 现场讲解话术

```text
OPC 不是没有人，而是用 AI 扩展一个人的执行半径。我本人负责目标和最终责任，AI Agent 分担策略、工程、业务生成和评估，形成轻量化创新单元。
```
