# 11 三个月补实路线

## 第 1 月：真实闭环和授权样本

目标：

- 完成 C1/C2/C3 三岗位真实闭环。
- 每个案例完成上传简历、能力诊断、简历润色、学习任务、至少 3 轮面试、报告、训练复盘和后台人工评分。
- 确认本次案例数据贡献授权。

验收物：

- CSV 和 Markdown 验收记录。
- 后台人工评分记录。
- 真实授权样本 manifest。

## 第 2 月：真实评测与 OpenAI SFT

目标：

- 准备真实授权样本和 holdout 集。
- 满足真实样本不少于 3、总训练样本不少于 10 的最低门槛，建议扩大到 30-50 条。
- 启动 OpenAI SFT 或先做 Prompt/RAG 对照 Eval。

验收物：

- `job_record.json` 或对照 Eval 报告。
- base model 与后续模型/策略的对比记录。
- 样本来源和脱敏记录。

## 第 3 月：高校场景小范围试点

目标：

- 找 1 个班级、社团或就业指导场景做小范围试用。
- 收集用户反馈和老师反馈。
- 形成场景对接方案、迭代计划和商业化假设。

验收物：

- 试点反馈表。
- 功能迭代清单。
- OPC 创业计划补充版。

## 边界

未完成真实 job、真实模型 ID、真实 holdout eval 前，不在材料中宣称“已完成真实微调”或“效果提升固定百分比”。
