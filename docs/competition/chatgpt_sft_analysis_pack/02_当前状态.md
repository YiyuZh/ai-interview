## 4. 当前阻塞点

### 4.1 Git 发布断点

本地 `main` 当前领先远端 `origin/main` 16 个提交。服务器通过 `git pull` 暂时拉不到最新 SFT 脚本和手册。

正式服务器开跑前，需要先在本地执行：

```bash
git push origin main
```

然后服务器才能执行：

```bash
cd /opt/apps/ai-interview
git pull
docker compose up -d --build
docker compose exec -T app alembic upgrade head
```

### 4.2 真实授权样本不足风险

创建 OpenAI SFT job 前，最低门槛是：

```text
真实授权样本 >= 3
总训练样本 >= 10
```

建议目标是：

```text
真实授权样本 >= 10
总训练样本 30-50
验证集 >= 5
```

如果真实授权样本不足，只能继续补真实案例，不能把构造样本包装成真实训练成果。

### 4.3 OpenAI 账号条件待确认

- 账号具备 fine-tuning 权限。
