# OPC 角色体系索引

## 1. 角色分层

| 层级 | Agent | 作用 |
|---|---|---|
| 总控层 | OPC Commander Agent | 人类总控，决定目标、边界、验收和责任 |
| 策略层 | Competition Research Agent / Defense Coach Agent | 赛事研究、答辩叙事、评委问答 |
| 工程层 | Codex Engineering Agent / Architecture Review Agent | 代码、脚本、部署、测试、架构复核 |
| 业务层 | Resume Evidence / Role Profile RAG / Gap / Resume Polish / Interviewer / Learning / Report | 就业诊断主链路 |
| 治理层 | Data Governance / Eval Judge / SFT Dataset | 授权、去标识化、评分、样本准入、后训练准备 |

## 2. 人机边界

```text
AI Agent 提供候选方案、自动化执行和初步评估；OPC 本人负责目标选择、真实性、隐私授权、评分标准、提交决策和最终答辩责任。
```

## 3. 答辩一句话

```text
我不是让一个 AI 包办项目，而是把多个 AI 角色放进可追踪、可验收、可复盘的协同链条中。
```
